🔄 Proceso

  1. Llama a payUDN() para obtener pagos y consumos de bitácora
  2. Llama a debtUDN() para obtener consumos pendientes
  3. Calcula: Deuda Total = debtUDN + payUDN['consumo']

🔍 Consultas SQL

1️⃣ payUDN() - Pagos y Consumos en Bitácora

SELECT 
    SUM(CASE WHEN Tipo = 'Pago' THEN Monto ELSE 0 END) AS pago,
    SUM(CASE WHEN Tipo = 'Consumo' THEN Monto ELSE 0 END) AS consumo
FROM creditos_bitacora cb
INNER JOIN creditos_udn cu ON cb.id_Credito = cu.id_Credito
WHERE cu.id_UDN = ?
AND cb.Fecha_Credito BETWEEN ? AND ?
AND cb.Stado = 1

Retorna:

  • pago: Total de pagos realizados
  • consumo: Total de consumos registrados en bitácora

2️⃣ debtUDN() - Consumos Pendientes

SELECT SUM(cc.Monto) AS deuda
FROM creditos_consumo cc
INNER JOIN creditos_udn cu ON cc.id_Credito = cu.id_Credito
WHERE cu.id_UDN = ?
AND cc.Fecha_Credito BETWEEN ? AND ?
AND cc.Stado = 1

Retorna:

  • deuda: Total de consumos pendientes de pago

📜 Reglas de Negocio

✅ Filtrado de Datos

  • Solo registros con Stado = 1 (activos)
  • Filtrar por UDN específica mediante tabla creditos_udn
  • Filtrar por rango de fechas en Fecha_Credito

💵 Formato de Salida

  • Todos los montos se formatean con evaluar()
  • Formato: "$ X,XXX.XX"
  • Siempre retornar estructura con claves debt y pay

🔗 Relaciones de Tablas

  • Usar INNER JOIN para relacionar clientes con UDN
  • Agregar datos con SUM() en SQL (no en PHP)
  • Usar CASE WHEN para separar tipos en bitácora

💻 Código PHP Simplificado

📄 _Clientes.php - payDebtUDN()

function payDebtUDN() {
    // 1. Obtener pagos y consumos de bitácora
    $payUDN = $this->payUDN([
        $this->udn,
        $this->date1,
        $this->date2
    ]);

    // 2. Obtener consumos pendientes
    $debtUDN = $this->debtUDN([
        $this->udn,
        $this->date1,
        $this->date2
    ]);

    // 3. Calcular deuda total
    $totalDebt = $debtUDN + $payUDN['consumo'];

    // 4. Formatear y retornar
    return [
        'debt' => evaluar($totalDebt),
        'pay'  => evaluar($payUDN['pago'])
    ];
}

📄 mdl-clientes.php - payUDN()

function payUDN($array) {
    $query = "
        SELECT 
            SUM(CASE WHEN Tipo = 'Pago' THEN Monto ELSE 0 END) AS pago,
            SUM(CASE WHEN Tipo = 'Consumo' THEN Monto ELSE 0 END) AS consumo
        FROM creditos_bitacora cb
        INNER JOIN creditos_udn cu ON cb.id_Credito = cu.id_Credito
        WHERE cu.id_UDN = ?
        AND cb.Fecha_Credito BETWEEN ? AND ?
        AND cb.Stado = 1
    ";
    
    $result = $this->_Read($query, $array);
    return $result[0];
}

📄 mdl-clientes.php - debtUDN()

function debtUDN($array) {
    $query = "
        SELECT SUM(cc.Monto) AS deuda
        FROM creditos_consumo cc
        INNER JOIN creditos_udn cu ON cc.id_Credito = cu.id_Credito
        WHERE cu.id_UDN = ?
        AND cc.Fecha_Credito BETWEEN ? AND ?
        AND cc.Stado = 1
    ";
    
    $result = $this->_Read($query, $array);
    return floatval($result[0]['deuda']);
}
